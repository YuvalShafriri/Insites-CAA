# Contributing

InSites-CAA is a research project by the **InSites Knowledge Lab**.

We welcome:

- **Issues** — bug reports, questions about the methodology, suggestions
- **Discussions** — ideas for new workflows, platform adaptations, use cases
- **Pull requests** — by arrangement; please open an issue first to discuss

If you adapt the CBSA methodology or bot prompts for your own domain, we'd love to hear about it.

**Contact:** Open an issue on this repository.

**License:** All contributions are under [CC-BY 4.0](LICENSE) — attribution to the InSites Knowledge Lab is required.
