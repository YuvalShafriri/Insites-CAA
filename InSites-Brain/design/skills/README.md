# Claude Code Skills

Reusable skills for Claude Code projects. To use: copy the skill folder into your project's `.claude/skills/` directory.

| Skill | Purpose |
|-------|---------|
| `lim/` | Less-Is-More — optimize any output for weight and density |
| `dashboard-ux-reviewer/` | Review dashboard UX using the 7-lens design framework |
| `expert-review/` | Launch parallel expert agents for multi-perspective review |
| `skill-creator/` | Create, test, and benchmark new skills |
