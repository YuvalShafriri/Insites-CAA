# Design System

Reusable design assets for InSites CBSA projects. Four categories:

| Directory | What's inside |
|-----------|--------------|
| `specs/` | Workflow specifications — how each bot feature works |
| `contracts/` | Cross-platform rules — visual language and epistemic notation |
| `examples/` | Reference implementations — working HTML dashboards |
| `skills/` | Claude Code skills — plug into any project |
